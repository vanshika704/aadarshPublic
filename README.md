🎓 Adarsh School Website

A modern, responsive school website platform designed to present academic information, infrastructure, and admission details while enabling easy content management through dedicated dashboards.

🚀 Overview

The Adarsh School Website is built to enhance the institution’s digital presence and provide a seamless experience for students, parents, and administrators.

It offers:

Clear presentation of academic and facility-related information
Easy access to admission details
Dynamic content updates via admin dashboard
✨ Features
📄 Informational pages (About, Facilities, Academics, Admissions)
📱 Fully responsive design (mobile + desktop)
🔐 Authentication system (Firebase-based)
🧑‍💼 Admin Dashboard for managing website content
👥 User Interface for students/parents to browse information
⚡ Real-time updates using Firebase
🧠 Key Highlights
Built with a focus on clean UI/UX and accessibility
Enables non-technical admins to update content easily
Designed for scalability and maintainability
Improves communication between school and users
🛠 Tech Stack

Frontend

React
Tailwind CSS

Backend / Services

Firebase (Authentication, Firestore/Realtime DB, Hosting)
🏗 Architecture
Component-based frontend using React
Firebase for backend services and real-time data handling
Role-based structure (Admin vs General User)
🔑 Admin Dashboard

The admin dashboard allows:

Updating school information
Managing content dynamically
Maintaining website without code changes

🎯 Purpose

This project aims to:

Digitize school information access
Simplify administrative content management
Provide a smooth and modern browsing experience
